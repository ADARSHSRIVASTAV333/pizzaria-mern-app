# 🍕 Pizzaria Ordering System

A full-stack MERN pizza ordering application with authentication, cart management, checkout flow, payment handling and order history.

## Features

- User Registration/Login
- JWT Authentication
- Password Encryption (bcrypt)
- Pizza Customization
- User-specific Shopping Cart
- Checkout Workflow
- Payment Page
- Order History
- MongoDB Integration

## Tech Stack

Frontend:
- React
- Bootstrap
- Axios

Backend:
- Node.js
- Express.js

Database:
- MongoDB

Authentication:
- JWT
- bcrypt

## Application Workflow

Register/Login
→ Order Pizza
→ Customize Pizza
→ Add To Cart
→ Shopping Cart
→ Checkout
→ Payment
→ My Orders

## Run Locally

Frontend:

cd pizzariaapp
npm install
npm start

Backend:

cd pizzariaserver
npm install
nodemon server.js
